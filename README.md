# hello.puppeteer-testing

Starter kit for automated browser testing of external sites using mocha and puppeteer.

## Usage

Add your tests, install dependencies and run tests

```bash
yarn install
npm test
```

Or with Docker

```bash
docker-compose run --rm test
```
